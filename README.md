# Life Helper Multi-Tool CLI

A Python-based command-line multi-tool utility application designed to provide useful daily tools in one place.

## Features

- Random Decision Maker
- Password Generator
- Unit Converter
- System Information Tool
- URL Cleaner & Validator
- Reminder / Timer
- Website Status Checker
- File Downloader
- JSON Data Manager
- Weather Checker (API Integration)

## Technologies Used

- Python
- APIs
- File Handling
- JSON
- Command-Line Interface (CLI)

## Purpose

This project was created to improve my Python programming skills and practice working with APIs, modular programming, system utilities, and command-line applications.

## How to Run

1. Clone the repository
2. Open the project folder
3. Run the main Python file
